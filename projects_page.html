PROJECTS

ELECTRONIC VOTING SYSTEM USING FINGERPRINT AUTHENTICATION

[Final-year project, ECE, UNN | 2018/2019]

Designed and Implemented a desktop solution for Electronic Voting using .Net Framework as part of my final year project in the University. The tools used included the Visual Studio, Visual Studio Code, Microsoft SQL Server, Mongo database, DigitalPersona SDK, Mozilla, Edge & Chrome Browsers. The code was authored in C# (logic) and the WPF (UI framework). I worked with a partner: Mr. Ekwedike Patrick.

The project allowed for the registeration of elections (i.e. candidates, parties, election time, election type) and the voter registeration. Voters fingerprints are captured at the point of registeration and used for authentication during the actual voting session. Elections can be monitored by the super-Admins and results are published on an associated web portal.

PERSONAL PORTFOLIO WEBSITE (http://www.odikanwamichael.epizy.com)

[Personal project | 12/08/2020 - 7/09/2020]

Designed, developed and deployed a highly responsive personal portfolio website. The development environment used was the Visual Studio Code and previews were made on Chrome, Mozilla and Edge browsers. The code was authored in HTML, CSS and javascript. The JQuery libary was also used. No frameworks and plugins were used.

The site is static: it does not collect any user input. It was designed with a mobile-first approach and then scaled to bigger screen sizes (most sizes of tablets and desktops). Code is properly documented for easy review and targeted at particular screen sizes.

All images used were either sourced personally or obtained freely online. They were also further compressed for better page-load time.
